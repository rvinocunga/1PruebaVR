Juego VR para PICO
Descripción
Juego para gafas VR PICO que ofrece dos modos de juego: un modo de un jugador con una galería de tiro a dianas y un modo multijugador con oleadas de enemigos.

Instalación
Para modificar el proyecto, se requiere Unity 2022.3.13f1.

Uso
La primera escena es el lobby, donde se puede seleccionar el modo de juego:

Modo de un Jugador: Consiste en hacer la máxima puntuación disparando a dianas que se mueven en un tiempo limitado.
Modo Multijugador: Se une automáticamente a una sala de servidor Photon, sin acabar...
